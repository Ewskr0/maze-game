# maze game
C++ School Project

## Get started

```bash
make
./maze <path to maze> [--ai]
```

You can use files located in `test/*.txt`
The option `--ai` activates the AI

## Features

* Manual player
* AI that resolves that maze
* No memory leak
* Modular code
